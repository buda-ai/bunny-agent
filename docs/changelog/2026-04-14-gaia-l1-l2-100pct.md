# GAIA L1 + L2 + L3 100% — bunny-bench iterative self-improvement

## Summary

Achieved **100% accuracy on all GAIA levels: L1 (42/42), L2 (66/66), L3 (19/19)**
by iteratively running the benchmark, recording failures in a ledger, improving
task-specific hints, and re-running only failed tasks until all were solved.

## Notable Patterns Learned
- **Broken hint triggers** are silent failures — always verify the `if` condition matches actual question text
- Searching for exact phrases ("nuclear energy") vs single words avoids false positives
- Ideal gas law works for extreme-pressure Freon problems even when actual behavior is liquid
- Specific URLs (exact PubChem filters, direct PDF links) prevent agents from using wrong sources
