---
title: Output Format
description: runner-cli output format behavior
---

The runner-cli no longer supports `--output-format`.
`sandagent run` always outputs AI SDK data stream (SSE) format.

## Usage

```bash
sandagent run [options] -- "<user input>"
```

## Data Stream Format

Outputs Server-Sent Events using the [AI SDK UI Stream Protocol](https://ai-sdk.dev/docs/ai-sdk-ui/stream-protocol).

```bash
sandagent run -- "Calculate 2+2"
```

```
data: {"type":"start","messageId":"msg_123"}
data: {"type":"text-start","id":"text_1"}
data: {"type":"text-delta","id":"text_1","delta":"The answer is 4."}
data: {"type":"text-end","id":"text_1"}
data: {"type":"finish","finishReason":"stop"}
data: [DONE]
```
