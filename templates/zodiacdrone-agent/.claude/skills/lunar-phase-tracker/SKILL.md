# Lunar Phase Tracker

This skill tracks lunar cycles, moon phases, and significant celestial events, providing detailed guidance on how each phase affects the zodiac signs and offering recommendations for activities aligned with lunar energy.

## When to Use This Skill

- Tracking current moon phase and upcoming changes
- Creating lunar phase content for the platform
- Providing moon-based guidance to users
- Planning content around new/full moons
- Alerting users to significant lunar events
- Generating lunar phase calendars

## What This Skill Does

1. **Phase Tracking**: Identifies current and upcoming moon phases
2. **Sign Transit**: Tracks which zodiac sign the moon is in
3. **Event Prediction**: Forecasts significant lunar events (eclipses, supermoons)
4. **Energy Analysis**: Interprets the energy of each lunar phase
5. **Activity Recommendations**: Suggests optimal activities for each phase
6. **Personalized Impact**: Explains how phases affect each zodiac sign
