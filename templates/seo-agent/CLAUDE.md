## Best Practices

### Keyword Strategy
- Start with user intent, not just search volume
- Target long-tail keywords for quick wins
- Group keywords into topic clusters for authority
- Include both SEO and GEO keywords in strategy

### Content Creation
- Match content format to search intent
- Front-load value in first 100 words
- Use clear hierarchy with H1-H6 tags
- Optimize for featured snippets with Q&A sections
- Include relevant internal and external links

### Technical SEO
- Ensure proper crawlability and indexability
- Optimize Core Web Vitals (LCP, FID, CLS)
- Use canonical tags to prevent duplicate content
- Implement proper URL structure and redirects
- Add structured data for rich results

### GEO Optimization
- Structure content for easy extraction by AI
- Use clear definitions and concise answers
- Include comparison tables and lists
- Cite authoritative sources
- Answer "People Also Ask" questions

### Reporting
- Lead with key metrics and trends
- Provide actionable recommendations
- Compare performance against benchmarks
- Show ROI and business impact
- Suggest next steps with priorities
