# Script Generation

## When to Use This Skill

- Need a timestamped script for product video creation
- Have a website URL and want to generate video narration
- Have product keywords or themes to develop into a script
- Need to expand a rough script idea into a full production script
- Want to create scripts optimized for 60-second product videos

## What This Skill Does

1. Analyzes product websites to extract key information
2. Processes product keywords and themes into narrative structure
3. Generates timestamped scripts suitable for video production
4. Optimizes script length for target video duration (typically 60 seconds)
5. Creates scripts with clear hooks, value propositions, and calls-to-action
6. Provides alternative versions for A/B testing

## How to Use

**From Website URL:**
```
Generate a video script for https://example.com
Target duration: 60 seconds
```

**From Keywords:**
```
Create a video script with these keywords: "fast", "secure", "cloud-based", "team collaboration"
Product type: Project management tool
```

**From Theme:**
```
Write a script about how AI improves customer support
Target audience: Small business owners
Duration: 60 seconds
```

**From Initial Script:**
```
Expand this script idea with timestamps:
"Our product helps teams collaborate better. It's fast, easy, and secure."
```

## Instructions

### Input Analysis Phase
1. Identify input type:
   - Website URL
   - Product keywords
   - Theme or topic
   - Initial script draft
2. If website URL provided, fetch and analyze content:
   - Extract product features
   - Identify value propositions
   - Note key differentiators
   - Capture brand tone
3. If keywords provided, understand product context
4. Determine target video duration (default: 60 seconds)
5. Understand target audience if specified

### Script Structure Planning
1. **Hook (0-3s)**: Attention-grabbing opening
   - Ask a compelling question
   - State a surprising fact
   - Present a common problem
2. **Problem Statement (3-15s)**: What pain point does this solve?
3. **Solution Introduction (15-30s)**: Introduce the product/service
4. **Key Features/Benefits (30-50s)**: Highlight 2-3 main benefits
5. **Call-to-Action (50-60s)**: Clear next step

### Script Generation
1. Write clear, conversational narration
2. Target 150-180 words for 60-second videos (2.5-3 words/second)
3. Use simple, direct language
4. Avoid jargon unless audience-appropriate
5. Include emotional hooks and value statements
6. Ensure brand consistency if website analyzed

### Timestamp Creation
1. Add timestamps for each script section
2. Format: `[00:00 - 00:03] Hook section text`
3. Mark key transition points
4. Note suggested visual cues
5. Include pacing guidelines

### Script Optimization
1. Read aloud to verify pacing
2. Check for difficult-to-pronounce phrases
3. Ensure clear pronunciation of product names
4. Verify logical flow and transitions
5. Confirm CTA is clear and actionable

### Output Format
```
# Video Script: [Product Name]
Duration: 60 seconds
Word Count: ~165 words
Target Platform: [Platform]

## Script with Timestamps

[00:00 - 00:03] Hook
"Are you tired of scattered project files and missed deadlines?"

[00:03 - 00:15] Problem
"Traditional project management tools are either too complex or too simple. Your team wastes hours switching between apps, and important updates get lost in email chains."

[00:15 - 00:30] Solution
"Introducing TaskFlow - the collaborative project management platform that actually understands how modern teams work."

[00:30 - 00:50] Features & Benefits
"With real-time sync, your entire team stays on the same page. Intuitive drag-and-drop makes planning effortless. And powerful analytics show you exactly where to focus your energy. TaskFlow brings everything together in one beautiful, easy-to-use workspace."

[00:50 - 00:60] Call-to-Action
"Start your free 30-day trial today. No credit card required. Join over 50,000 teams who've already streamlined their workflow with TaskFlow."

## Visual Cues
- 00:00-00:03: Show cluttered desktop, frustrated user
- 00:03-00:15: Split screen: chaos vs. order
- 00:15-00:30: Product logo reveal, dashboard overview
- 00:30-00:50: Feature demonstrations (sync, drag-drop, analytics)
- 00:50-00:60: CTA screen with website/signup button

## Pronunciation Guide
- TaskFlow: TASK-flow (emphasis on first syllable)

## Alternative Hooks (for A/B testing)
1. "What if managing projects felt effortless?"
2. "Your team deserves better than spreadsheets and email."
3. "There's a smarter way to manage projects."
```

## Advanced Features

- **Multi-language scripts**: Generate scripts in different languages
- **Industry-specific templates**: E-commerce, SaaS, B2B, B2C variations
- **Tone adjustment**: Professional, casual, inspiring, urgent
- **Length variations**: 30s, 60s, 90s versions
- **A/B testing hooks**: Multiple opening variations

## Tips for Success

1. **Word count accuracy**: Aim for 2.5 words per second (150 words = 60s)
2. **Front-load value**: Most viewers decide in first 3 seconds
3. **Conversational tone**: Write like you're talking to a friend
4. **Specific benefits**: "Save 10 hours per week" beats "Save time"
5. **Clear CTA**: Tell viewers exactly what to do next
6. **Visual thinking**: Write for the eye and ear together
