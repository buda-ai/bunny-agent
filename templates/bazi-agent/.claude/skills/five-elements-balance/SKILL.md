# Five Elements Balance Analysis

This skill analyzes the distribution of the Five Elements (五行 Wǔ Xíng) in a BaZi chart to understand overall balance, identify elemental deficiencies or excesses, and provide insights into life areas, health, and harmony.

## When to Use This Skill

- **After BaZi calculation**: Core analysis step
- **Health consultation**: Elemental balance affects organ systems
- **Career guidance**: Match work to strong/weak elements
- **Feng Shui advice**: Identify elements to enhance in environment
- **Name selection**: Determine which elements to strengthen
- **Lifestyle recommendations**: Colors, directions, foods to balance

## What This Skill Does

1. **Element Counting**: Tallies all elements from stems, branches, and hidden stems
2. **Balance Assessment**: Identifies strong, weak, or missing elements
3. **Relationship Analysis**: Examines generating (生) and controlling (克) cycles
4. **Life Area Mapping**: Connects elements to career, health, relationships, etc.
5. **Balancing Recommendations**: Suggests how to harmonize elemental energy

## The Five Elements System

### Generating Cycle (相生 Xiāng Shēng)
```
Water → Wood → Fire → Earth → Metal → Water
 💧    🌳    🔥    ⛰️    ⚡
```

- **Water generates Wood**: Nourishes growth
- **Wood generates Fire**: Fuels burning
- **Fire generates Earth**: Creates ash/soil
- **Earth generates Metal**: Forms minerals
- **Metal generates Water**: Condenses moisture

### Controlling Cycle (相克 Xiāng Kè)
```
Water controls Fire
Fire controls Metal
Metal controls Wood
Wood controls Earth
Earth controls Water
```

## Analysis Process

### Step 1: Count Elements

Count from three sources:

**Heavenly Stems (天干)** - Full strength
```
Year Stem: 己 (Earth)
Month Stem: 丁 (Fire)
Day Stem: 甲 (Wood)
Hour Stem: 癸 (Water)
```

**Earthly Branches (地支)** - Primary element
```
Year Branch: 巳 (Fire)
Month Branch: 丑 (Earth)
Day Branch: 子 (Water)
Hour Branch: 酉 (Metal)
```

**Hidden Stems (藏干)** - Secondary strength
```
巳: 庚(Metal), 丙(Fire), 戊(Earth)
丑: 己(Earth), 癸(Water), 辛(Metal)
子: 癸(Water)
酉: 辛(Metal)
```

### Step 2: Calculate Totals

Example count:
```
Wood (木): 1 stem = 1.0 point
Fire (火): 1 stem + 1 branch + 1 hidden = 2.5 points
Earth (土): 1 stem + 1 branch + 2 hidden = 3.0 points
Metal (金): 1 branch + 3 hidden = 2.5 points
Water (水): 1 stem + 1 branch + 2 hidden = 3.0 points
━━━━━━━━━━━━━━━━━━━━━━━━━━━
Total: 12.0 points
```

### Step 3: Calculate Percentages
```
Wood: 8.3%
Fire: 20.8%
Earth: 25.0%
Metal: 20.8%
Water: 25.0%
```

### Step 4: Assess Balance

**Balanced Chart**: Each element 15-25%
**Strong Element**: >30%
**Weak Element**: <10%
**Missing Element**: 0%

### Step 5: Analyze Relationships

- Does Day Master have support (generating element present)?
- Is Day Master being controlled (controlling element excessive)?
- Are elements in harmonious cycles?
- Are there element clusters or voids?

## Best Practices

### Interpretation Guidelines
- Consider **percentages AND relationships**, not just counts
- Assess strength **in context of season** and Day Master
- Look for **cycles** - are elements flowing or blocked?
- Identify **priority imbalances** to address first
- Frame **as opportunities**, not deficiencies

### Communication
- Use accessible metaphors (not just technical terms)
- Explain **why** balance matters (real-life impact)
- Provide **actionable** recommendations
- Balance **traditional wisdom** with modern lifestyle
- Encourage **gradual changes**, not overwhelming overhaul

### Avoid
- ❌ Oversimplifying complex interactions
- ❌ Ignoring seasonal context
- ❌ Deterministic health predictions
- ❌ Impractical recommendations (e.g., "move to Water country")
- ❌ Creating anxiety about imbalances
